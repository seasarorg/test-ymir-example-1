セッションタイムアウト

セッションタイムアウトのサンプルです。

ログイン中にセッションがタイムアウトすると、通常はセッションからログイン情報が取得できなくなって権限エラーが発生しますが、未ログインのケースと、ログインしていたがセッションがタイムアウトしたケースを区別したい時があります。

このサンプルでは、ログインしていない状態でログイン後の画面にアクセス（以下のフォームでログインする前に「ログインが必要な画面」リンクをクリック）すると権限エラー、ログインしていたがセッションがタイムアウトした場合にログイン後の画面にアクセス（以下のフォームでログインした後に一定時間経過してから「ログインが必要な画面を再度表示する」をクリック）するとタイムアウトエラーが発生します。
